# UniHub: Smart Resource Sharing Platform for Students

UniHub is a web-based Student Resource Hub where students can register, log in, browse and upload study resources organized by category, bookmark useful materials, view notices, and submit feedback through a contact form.

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- PHP 8
- MySQL
