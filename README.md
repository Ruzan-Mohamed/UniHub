# UniHub: Smart Resource Sharing Platform for Students

UniHub is a web-based Student Resource Hub where students can register, log in, browse and upload study resources organized by category, bookmark useful materials, view notices, and submit feedback through a contact form.

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- PHP 8
- MySQL

## Setup Instructions

### Prerequisites
Before running the project, ensure the following software is installed:

- XAMPP (Apache & MySQL)
- PHP 8 or later
- MySQL
- Git (optional, for cloning the repository)
- A modern web browser (Google Chrome, Microsoft Edge, Firefox, etc.)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/Ruzan-Mohamed/UniHub.git
   ```

2. Move the project folder to the XAMPP `htdocs` directory.

3. Start **Apache** and **MySQL** using the XAMPP Control Panel.

4. Open **phpMyAdmin**.

5. Create a database named:

   ```
   unihub
   ```

6. Import the `database.sql` file into the `unihub` database.

7. If necessary, update the database connection settings in:

   ```
   includes/db.php
   ```

8. Open your browser and visit:

   ```
   http://localhost/UniHub/
   ```
